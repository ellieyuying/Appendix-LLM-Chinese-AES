# 大语言模型在汉语作文自动评估中的可解释性研究 —— 附录

本仓库包含以下论文的附录材料：

> 王予沛、张钰莹、胡韧奋．大语言模型在汉语作文自动评估中的可解释性研究[J]．DOI：10.3969/j.issn.2095-798X.2026.01.002

## 附录内容概览

### A. 大模型提示词

包含本研究中使用的全部提示词设计，涵盖以下场景：

1. **作文评分提示词（零样本学习）**：指令模型依据 HSK 12 级评分标准，以 JSON 格式输出评分结果。
2. **作文评分提示词（小样本学习）**：提供 5 篇示例作文后，指令模型参照示例进行评分。
3. **语言复杂化改写提示词**：指令模型在保持原文核心内容不变的前提下，通过引入高级词汇、成语及复杂句式来提升作文语言复杂度。
4. **语言简化改写提示词**：指令模型将作文改写为使用基础词汇与简单句式的版本，以降低语言复杂度。
5. **作文反馈提示词（通用版）**：通过两轮对话，先完成评分，再从语言准确性、语言复杂性、篇章组织水平三个维度生成结构化反馈。
6. **作文反馈提示词（文体专项版）**：分别针对记叙文与议论文的文体特点，设计了差异化的反馈生成提示词。

### B. 反事实干预构造方式

描述了本研究中八类反事实干预策略的具体实现方法，按干预维度分为：

- **语言准确性干预**（4 类）：错误校正（+）、书写错误引入（−）、语法错误引入（−）、词序错误引入（−）
- **篇章组织水平干预**（2 类）：段内句序重排（−）、文内句序重排（−）

> 注：语言复杂性干预（语言复杂化 + / 语言简化 −）通过附录 A 中的大模型改写提示词实现。

### C. 反事实样本的有效性评估

通过词汇数量、句子数量、平均句长、平均依存树深度、词汇多样性（TTR）、词汇复杂度六项语言学指标，以及语义余弦相似度，验证了错误校正、语言复杂化、语言简化三类干预所生成反事实样本的有效性（效果量采用 Cohen's D 度量）。

### D. 不符合干预预期的反馈模式

提供了两类典型的不符合预期反馈案例：

- **语言简化干预案例**：展示了语言简化在修正语言自然度的同时意外提升评分和反馈质量的现象。
- **篇章组织水平干预案例**：展示了当原始作文篇章问题已较严重时，模型无法在反馈中识别干预带来的进一步变化的情况。
